Laravel KA

Laravel (5.x) Georgian Language Pack.

Installation

  1. Download the archive and place the ka folder into your /resources/lang/ directory.
  2. Open /config/app.php file, replace locale key from en to ka and you're ready to go.
'locale' => 'ka',

Validation Translation

If you want to translate form field names, during validation, open the /resources/lang/ka/validation.php and at the end of the file set up the attributes array:

'attributes' => [
    'my_title' => 'სათაური'
],

In that case, if you have required rule on the my_title field and you'll leave blank during form submit, you'll get the validation error message სათაური აუცილებელია instead of my_title აუცილებელია.
